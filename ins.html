Instructions

  • The objective is to make all obstacles disappear in the given time limit in order to advanced to new level.
  • The obstacles will disappear when the "bird"comes in proximity of it.
  • The number of birds is limited.The player has to remove all obstacles and advance using the given number of words.
  • If the time runs out the player runs out of birds,the game ends.
  • When the game ends, the player has option to restart the game.
